# Digital Griot: AI-Powered Emotion-Adaptive Interactive Storytelling Framework

Digital Griot is an AI-powered interactive storytelling system designed to generate expressive narration using emotion-aware text-to-speech techniques. The system analyzes story text, detects emotional context, and adapts voice narration to produce natural and engaging storytelling.

## Project Overview

Traditional storytelling systems often generate speech with static or predefined emotions, resulting in unnatural narration and poor emotional continuity. Digital Griot addresses this limitation by detecting emotions from story text and dynamically adapting narration. The framework integrates Natural Language Processing (NLP) and speech synthesis techniques to maintain emotional flow across the entire story.

## Key Features

- Emotion detection from story text using transformer-based models  
- Adaptive voice generation based on emotional context  
- Smooth emotional transitions across story segments  
- Modular architecture for flexible system design  
- Improved storytelling engagement and narrative coherence  

## System Architecture

The Digital Griot framework consists of the following modules:

1. Text Segmentation  
2. Emotion Detection using NLP models  
3. Emotion Transition Modelling  
4. Adaptive Voice Selection  
5. Text-to-Speech Synthesis  
6. Narration Output Generation  

These modules work together to analyze story content and produce expressive narration with emotional continuity.

## Technologies Used

- Python  
- Natural Language Processing (NLP)  
- Transformers (DistilBERT)

- Machine Learning

## Results

The proposed system was implemented as a functional prototype and tested with multiple storytelling scenarios. Results demonstrate improved emotional continuity, adaptive voice narration, and better storytelling engagement compared with conventional text-to-speech systems.
